## 0.1.0

Initial release.

- Push-to-talk dictation via Whisper (tiny, base, small, medium, large)
- GPU acceleration (Vulkan on Windows/Linux, Metal on macOS)
- Output modes: paste at cursor or copy to clipboard
- Configurable hotkeys for record and output toggle
- Status overlay with position, size, and monitor selection
- Model management: download, delete, loading strategies (eager, lazy, per-use)
- Language selection (auto-detect + 11 languages)
- Audio input device selection
- Transcription history with retention settings
- System tray with quick access
- Autostart support
- File transcription
- Windows, macOS, Linux
